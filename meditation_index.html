    Final Writeup: Heart Rate Trends During Meditation

    Motivation

    The goal of this visualization was to explore how heart rate patterns vary across different meditation techniques, and whether factors like gender identity and session type influence these trends. Meditation is often associated with physical and emotional regulation, and we wanted to provide a visual exploration of these effects that is both data-driven and personally engaging.

    Visualization Design

    Our visualization combines two complementary approaches:

    • Line Chart: Shows detailed heart rate fluctuations over time, enabling temporal analysis while revealing person-specific variation.
    • Human Figures with Beating Hearts: Provides an intuitive and engaging representation of heart rates through animated figures, where the beating animation corresponds to actual BPM values.

    Interactive Elements

    The interface includes:

    • Technique & gender filters that update both visualizations in real time
    • Time slider to animate through the meditation session
    • Play/pause/reset controls for the animation
    • Speed control for animation playback
    • Interactive tooltips that provide detailed information on hover

    Development Process

    We developed this visualization using D3.js for data manipulation and SVG creation, with several key steps:

    1. Data preprocessing to identify meditation techniques and structure
    2. Line chart implementation for temporal analysis
    3. Human figure SVG design for each meditation technique
    4. Heart beat animation implementation with variable rates
    5. Time slider and control integration
    6. Responsive design and cross-browser testing

    Insights & Findings

    Through this visualization, users can discover:

    • How different meditation techniques affect heart rate patterns
    • Gender differences in physiological responses to meditation
    • The progression of heart rate changes throughout a meditation session
    • Techniques that produce the most significant heart rate reductions
